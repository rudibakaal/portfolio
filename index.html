Detecting SMS Spam with Neural Networks

In this python project, I implemented a neural network to classify SMS messages as spam or not using the SMS Spam Collection dataset. Employed the TfidfVectorizer to transform messages into term-document matrices, and used one-hot encoding for binary labels. The model, utilizing a binary-crossentropy loss function and the Nadam optimizer, achieved impressive 98% and 97% accuracy for binary and validation classifiers respectively, after 35 epochs.
